## Основная логика

Тест состоит из 5 шагов:

1. Тип кожи
2. Особенности кожи
3. Образ жизни
4. Использование SPF
5. Формат средств

На шаге «Особенности кожи» пользователь может выбрать несколько вариантов. Для подбора используется только одна приоритетная особенность в порядке:

Купероз → Розацеа → Пигментация → Акне / высыпания → Чувствительная / раздражённая → Без особенностей

## Результат

После прохождения теста пользователь видит:

- краткий профиль по ответам;
- объяснение выбранного приоритета;
- основную рекомендацию;
- альтернативные средства, если они подходят по выбранному формату;
- профессиональный вариант, если он нужен;
- дисклеймер о консультации с врачом.

## Аналитика

Проект должен сохранять данные о прохождении теста:

- дату и время прохождения;
- выбранный тип кожи;
- выбранные особенности кожи;
- приоритетную особенность;
- образ жизни;
- сценарий использования SPF;
- выбранный формат косметики;
- показанную основную рекомендацию;
- показанные альтернативы;
- показанный профессиональный вариант;
- источник запуска, если тест открыт из Telegram, MAX или сайта.
